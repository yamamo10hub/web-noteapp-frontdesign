# noteapp

## 概要
作業時に使うメモ帳が欲しかった  
* 行をハイライトできるとうれしい
* ctrl+cが手間１（カーソルによる範囲選択以外で行選択したい）
* ctrl+cが手間２（押下せずマウスクリックでコピーしたい）
* 多少色分けされてると良い
上記を満たすフリーテキストエディタが見つからない  
  
htmlで作ることを考えた。  
その場限りでは虚しいのでアプリ開発のフロント作成の真似事をしてみる。  
  
## 工程
アプリ側なにも決まって無いのでフロントだけ当repositoryで作る  
html + css + js でそれっぽく作れそうなのでできるとこまで試す
